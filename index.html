Coalition

About Coalition

Coalition is an open code project that allows front-end developers to share modules in their apps immediately, leveraging webpack module federation. The project's goal is to create an experience similar to Docker, but for front-end developers.

  • Allow developers to dynamically run code from multiple different sources in a single web application.
  • Enable the sharing of dependencies between different applications, reducing the overall load and initialization time.
  • Allow for the isolation of code, meaning that different parts of an application (or different applications altogether) can use different versions of the same dependency without conflict.
  • Supports both synchronous and asynchronous loading of remote modules, providing flexibility in how applications are structured and loaded.
  • Facilitate of micro-frontends. Different teams can work on different parts of an application independently, and integrate their parts into a larger application.
  • Hot Update remote modules. Update with new features or fixes without requiring a full redeployment of the entire application.
